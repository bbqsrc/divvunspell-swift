# DivvunSpell-Swift

A Swift wrapper around DivvunSpell.

## Example

To run the example, use `cargo-cocoapods`.

```
cargo pod example
```

## Installation

DivvunSpell is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'DivvunSpell'
```
